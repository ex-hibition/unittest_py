## git clone

- ローカルマシン 
```
$ git clone http://atlassian.nifty.com/stash/scm/sknext/test_sample_py.git
```

## 実行方法

- サンプルコード実行
```
$ python sample.py
```

- テストスイート実行
```
$ python -m unittest tests.test_sample
```
